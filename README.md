# TaskFoundry

TaskFoundry is a reusable LLaMA-Factory distillation template for building
domain-adapted small models with:

- teacher LoRA tuning
- response-based distillation
- offline packaging
- automated JSON / Markdown / Excel evaluation

Project sources live in [`llm_distill_law/`](./llm_distill_law/). The folder name
is kept for local compatibility, but the codebase itself has been cleaned into a
generic public template.
